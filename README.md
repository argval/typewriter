# Typewriter

A modern, desktop-based notebook application built with Next.js and Tauri. Create, edit, and manage your notes with a beautiful and intuitive interface.

## Features

- 📝 Multiple cell types (Code, Markdown, Drawing)
- 📚 Notebook management with sidebar
- 🎨 Beautiful and responsive UI
- 🖥️ Cross-platform desktop application

## Development

1. Install dependencies:
```bash
bun install
```

2. Run the development server:
```bash
bun run tauri dev
```

## Building

To create a production build:
```bash
bun run tauri build
```

## Tech Stack

- Next.js
- Tauri
- TypeScript
- Tailwind CSS
- shadcn/ui
